## Install the extension

Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/tabwright/dkfhphbajbkplddmchbdgdddioonngep), then click the extension icon on any tab. The icon turns **green** when connected.

The extension works with Chrome, Chromium, Brave, Arc, Edge, and any Chromium-based browser.

<Aside>
<Note>
Only tabs where you click the extension icon are controllable. Other tabs stay private.
</Note>
</Aside>

## Install the CLI

```bash
npm i -g tabwright
```

Or use without installing:

```bash
npx tabwright@latest session new
```

**Requirements:** Node.js 18+.

## Install the skill

The **skill** teaches your AI agent how to use Tabwright: which selectors to use, how to read snapshots, how to avoid common mistakes, and all available utilities. Import the `skills/tabwright` directory with your agent's official Agent Skills-compatible manager.

Skill installation, discovery, and updates belong to that manager. Tabwright does not write into agent-specific skill directories; the CLI remains the browser and capability runtime.

## Quick start

```bash
tabwright session new                    # creates sandbox, outputs id (e.g. 1)
tabwright -s 1 -e 'await page.goto("https://example.com")'
tabwright -s 1 -e 'console.log(await snapshot({ page }))'
tabwright -s 1 -e 'await page.locator("aria-ref=e5").click()'
```

## Start Chrome for Testing

If you don't have Chrome running or want a clean instance with the extension pre-loaded:

```bash
tabwright browser start
```

This auto-finds **Chrome for Testing** or Chromium on your system, launches it with recording flags enabled, and loads the bundled Tabwright extension. You can also pass a specific browser binary:

```bash
tabwright browser start /path/to/chrome
```

## Verify it works

After installing the extension and CLI:

1. Open any website in Chrome
2. Click the Tabwright extension icon (turns green)
3. Run:

```bash
tabwright session new
tabwright -s 1 -e 'console.log(await page.title())'
```

If you see the page title printed, everything is working.
