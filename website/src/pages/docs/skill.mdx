This page is the **full agent reference** for Tabwright. It's intended for AI agents and coding assistants that need to know every available utility, pattern, and best practice. If you're a human looking for a quick overview, start with [Installation](./installation.mdx) or [CLI Reference](./cli.mdx) instead.
</Note>

The Skill manager owns discovery, installation, and updates; Tabwright provides the runtime only.

The skill gives the agent the content below so it can use Tabwright without trial and error. It works with agents that support the standard skill structure.
