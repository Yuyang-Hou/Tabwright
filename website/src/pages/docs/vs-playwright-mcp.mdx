Playwright MCP is the official browser tool server from the Playwright team. It can launch a managed
browser, connect over CDP, or use its browser extension to work with existing signed-in tabs.

Tabwright also provides live browser control, but its product boundary extends beyond the current agent
session: it captures successful work, records the verified execution strategy and safety contract, and
exports the result as a portable Agent Skill.

## Comparison

|  | Playwright MCP | Tabwright |
|---|---|---|
| Existing signed-in Chrome | Supported with extension mode | Supported with per-tab extension control |
| Primary abstraction | Browser tools for the current agent task | Browser runtime plus reusable Agent Skills |
| Reusable workflow contract | Agent prompt or external code | Inputs, outputs, permissions, side effects, auth, execution strategy |
| Protected interactions | Agent operates the page | Skill can use UI trigger + network result as a hybrid workflow |
| Human verification | Handled by the calling agent | Declared `needs_human` checkpoint with resume guidance |
| Distribution | MCP configuration | Portable Agent Skill for compatible managers |
| Low-level access | Playwright browser tools | Full Playwright execution and raw CDP helpers |

## The difference is what survives the session

Browser tools help an agent complete one task. A Tabwright capability is successful only when a fresh
agent session can route to the saved Skill, provide structured input, respect its confirmation boundary,
and complete the same work without rediscovering the site.

## How Tabwright is different

Tabwright treats direct HTTP calls as an optimization, not the product goal. If a website needs its own
frontend to produce a dynamic signature, the Skill remains browser-backed. It can trigger the legitimate
request through the UI, observe its response, and return structured output. CAPTCHA and two-factor screens
become explicit human checkpoints instead of generic automation failures.

```bash
tabwright replay make <replay-id> protected-generation --goal "generate an asset" --force --json
tabwright capability describe protected-generation --json
# execution.strategy: hybrid
# execution.requiresUserBrowser: true
# execution.humanAssistance: on-challenge
```
